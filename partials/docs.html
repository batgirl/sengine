API Documentation

Our API allows developers to seamlessly leverage Sengine.io's REPL and hosting services in their applications through simple HTTP requests.

Execution

Execute JavaScript, Ruby, and Python code in its optimal runtime engine. This may be used to power browser and mobile text editors, multiplayer code wars, blog tutorials, and more.

http://sengine.io/api/v1/execute
{
  "data": //code goes here as a string 
}

Hosting

Instantly deploy and host static content for thirty minutes. This may be used to power web eccentric text editors, drag-and-drop HTML/CSS/JS applications, or even to preview your own applications in a production environment.

http://sengine.io/api/v1/host
{
  "data": //code goes here as a string
}

Integration

Provided below are examples of API calls to our service that utilize HTTP requests for different front end frameworks and libraries.

jQuery

Using $.ajax.

$.ajax({
  type: "POST",
  url: "http://sengine.io/api/v1/execute",
  data: {
    "data": //code goes here as a string
  }
});
$.ajax({
  type: "POST",
  url: "http://sengine.io/api/v1/host",
  data: {
    "data": //code goes here as a string
  }
});

AngularJS

Using the $http provider in a custom factory.

app.factory('httpFactory', function ($http) {
  return {
    execute: execute,
    hostHTML: hostHTML
  }
  function execute(code) {
    return $http.post('http://sengine.io/api/v1/execute', 
      {
        "data": //code goes here as a string
      });  
  }
  function hostHTML(code) {
    return $http.post('http://sengine.io/api/v1/host', 
      {
        "data": //code goes here as a string
      });
  }
});

ReactJS

Using $.ajax from inside component methods.

var ComponentExample = React.createClass({
  ...
  handleExecuteSubmit: function(code) {
    $.ajax({
      url: this.props.executeUrl,
      dataType: 'json',
      type: 'POST',
      data: //code goes here as a string,
      success: function(data) {
        this.setState({data: data});
      }.bind(this),
      error: function(xhr, status, err) {
        console.error(this.props.url, status, err.toString());
      }.bind(this)
    });
  },
  handleHostSubmit: function(code) {
    $.ajax({
      url: this.props.hostUrl,
      dataType: 'json',
      type: 'POST',
      data: //code goes here as a string,
      success: function(data) {
        this.setState({data: data});
      }.bind(this),
      error: function(xhr, status, err) {
        console.error(this.props.url, status, err.toString());
      }.bind(this)
    });
  },
  ...
});

ReactDOM.render(
  <ComponentExample executeUrl="http://sengine.io/v1/api/execute" 
  hostUrl="http://sengine.io/v1/api/host" />,
  document.getElementById('content')
);
